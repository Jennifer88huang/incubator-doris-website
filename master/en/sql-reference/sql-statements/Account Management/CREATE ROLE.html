# CREATE ROLE

# Description

The statement user creates a role

Grammar: CREATE ROLE role1;

This statement creates an unauthorized role that can be subsequently granted permission through the GRANT command.

# example

  1. Create a role

CREATE ROLE role1;

# keyword

CREATE, ROLE
