# SHOW DELETE

# Description

This statement is used to show successful historical delete tasks performed Grammar: SHOW DELETE [FROM db_name]

# example

  1. Show all historical delete tasks for database SHOW DELETE FROM database;

# keyword

SHOW,DELETE
