# UNINSTALL PLUGIN

# description

To uninstall a plugin.

Syntax

    UNINSTALL PLUGIN plugin_name;
    
    plugin_name can be found by `SHOW PLUGINS;`.
    
    Can only uninstall non-builtin plugins.

# example

1. Uninstall a plugin:

    UNINSTALL PLUGIN auditdemo;

# keyword

UNINSTALL,PLUGIN
