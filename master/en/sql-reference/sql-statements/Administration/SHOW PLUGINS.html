# SHOW PLUGINS

# description

To view the installed plugins.

Syntax

    SHOW PLUGINS;
    
    This command will show all builtin and custom plugins.

# example

1. To view the installed plugins:

    SHOW PLUGINS;

# keyword

SHOW PLUGINS
