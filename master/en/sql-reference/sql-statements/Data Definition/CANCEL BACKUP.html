# CANCEL BACKUP

# Description

This statement is used to cancel an ongoing BACKUP task. Grammar: CANCEL BACKUP FROM db_name;

# example

  1. Cancel the BACKUP task under example_db. CANCEL BACKUP FROM example_db;

# keyword

CANCEL, BACKUP
