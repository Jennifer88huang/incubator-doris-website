# reverse

# description

# Syntax

VARCHAR reverse(VARCHAR str)

The REVERSE() function reverses a string and returns the result.

# example

mysql> SELECT REVERSE('hello');
+------------------+
| REVERSE('hello') |
+------------------+
| olleh            |
+------------------+
1 row in set (0.00 sec)

mysql> SELECT REVERSE('你好');
+------------------+
| REVERSE('你好')   |
+------------------+
| 好你              |
+------------------+
1 row in set (0.00 sec)

# keyword

REVERSE
