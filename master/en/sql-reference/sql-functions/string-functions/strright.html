# strright

# Description

# Syntax

'VARCHAR strright (VARCHAR str)'

It returns the right part of a string of specified length, length is char length not the byte size.

# example

mysql> select strright("Hello doris",5);
+-------------------------+
| strright('Hello doris', 5) |
+-------------------------+
| doris                   |
+-------------------------+

# keyword

STRRIGHT
