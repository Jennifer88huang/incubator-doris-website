# left

# Description

# Syntax

'VARCHAR left (VARCHAR str)'

It returns the left part of a string of specified length, length is char length not the byte size.

# example

mysql> select left("Hello doris",5);
+------------------------+
| left('Hello doris', 5) |
+------------------------+
| Hello                  |
+------------------------+

# keyword

LEFT
