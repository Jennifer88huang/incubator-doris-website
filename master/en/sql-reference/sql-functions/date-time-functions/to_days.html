# to_days

# Description

# Syntax

'INT TO DAYS'

Days of returning date distance 0000-01-01

The parameter is Date or Datetime type

# example

mysql> select to_days('2007-10-07');
+-----------------------+
| to_days('2007-10-07') |
+-----------------------+
|                733321 |
+-----------------------+

##keyword TO_DAYS,TO,DAYS
