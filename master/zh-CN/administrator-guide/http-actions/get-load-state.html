# GET LABEL STATE

# description

NAME:
    get_load_state: get load's state by label
    
SYNOPSIS
    curl -u user:passwd http://host:port/api/{db}/get_load_state?label=xxx

DESCRIPTION
    Label

RETURN VALUES
    Json
    Label label uuid
    StatusSuccess
    Message 
    State: StatusSuccess
       UNKNOWN: Label
       PREPARE: prepare
       COMMITTED: cancel
       VISIBLE: cancel
       ABORTED: ROLLBACK
    
ERRORS

# example

1. testDb, testLabel
    curl -u root http://host:port/api/testDb/get_load_state?label=testLabel

# keyword

GET, LOAD, STATE
