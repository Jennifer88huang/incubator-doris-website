# Query Profile Action

# Request

GET /rest/v1/query_profile/<query_id>

# Description

Query Profile Action 用于获取 Query 的 profile。

# Path parameters

  • <query_id>

    可选参数。当不指定时，返回最新的 query 列表。当指定时，返回指定 query 的 profile。

# Query parameters

无

# Request body

无

# Response

  • 不指定 <query_id>
