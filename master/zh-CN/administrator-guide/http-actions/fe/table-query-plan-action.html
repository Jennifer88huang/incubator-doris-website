# Table Query Plan Action

# Request

POST /api/<db>/<table>/_query_plan

# Description

给定一个 SQL，用于获取该 SQL 对应的查询计划。

该接口目前用于 Spark-Doris-Connector 中，Spark 获取 Doris 的查询计划。

# Path parameters

  • <db>

    指定数据库

  • <table>

    指定表

# Query parameters

无

# Request body

{
	"sql": "select * from db1.tbl1;"
}
