# length

# description

# Syntax

INT length(VARCHAR str)

返回字符串的字节。

# example

mysql> select length("abc");
+---------------+
| length('abc') |
+---------------+
|             3 |
+---------------+

mysql> select length("中国");
+------------------+
| length('中国')   |
+------------------+
|                6 |
+------------------+
