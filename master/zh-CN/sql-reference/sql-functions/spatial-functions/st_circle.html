# ST_Circle

# description

# Syntax

GEOMETRY ST_Circle(DOUBLE center_lng, DOUBLE center_lat, DOUBLE radius)

将一个WKT（Well Known Text）转化为地球球面上的一个圆。其中center_lng表示的圆心的经度， center_lat表示的是圆心的纬度，radius表示的是圆的半径，单位是米,最大支持9999999

# example

mysql> SELECT ST_AsText(ST_Circle(111, 64, 10000));
+--------------------------------------------+
| st_astext(st_circle(111.0, 64.0, 10000.0)) |
+--------------------------------------------+
| CIRCLE ((111 64), 10000)                   |
+--------------------------------------------+
