# DROP CLUSTER

# description

该语句用于删除逻辑集群，成功删除逻辑集群需要首先删除集群内的db，需要管理员权限

语法

DROP CLUSTER [IF EXISTS] cluster_name

# example

删除逻辑集群 test_cluster

DROP CLUSTER test_cluster;

# keyword

DROP,CLUSTER
